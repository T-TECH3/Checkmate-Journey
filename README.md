  <h1>Checkmate Journey</h1>
</div>

Checkmate Journey is a maze and dungeon-crawler-style video game coded in C++. The game is played in the terminal and utilizes Unicode characters and color to create nice visuals. The game is played with an Xbox 360 Controller and uses the SDL2 [library](https://www.libsdl.org/).

This game is a great example of the following coding principals demonstrated in practical use cases:

- Classes
- Inheritance
- Polymorphism
- Linked Lists
- Stacks
- Queues
- Object Oriented Programming Principles & Conventions
- Pointers
- Maze Solving Algorithm
- Unicode Encoding
- Controller Integration
- Cursor Movement
